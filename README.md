- 애완식물 마리모를 등록하면 D-day, 사진과 함께 성장 일기를 기록할 수 있어요.
- 마리모에게 물과 먹이를 주고 어플에 기록하면 다음 물과 먹이를 줄 날짜를 푸시 알림으로 알 수 있어요.
- 귀여운 마리모가 현재 날씨를 온도와 이미지로 알려줘요.
